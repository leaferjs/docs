# corner

路径圆角。

支持曲线与折线、折线与曲线之间的圆角。

## 安装插件

需要安装 corner 插件才能使用，[点此访问 Github 仓库](https://github.com/leaferjs/leafer-in/tree/main/packages/corner)。

::: code-group

```sh [npm]
npm install @leafer-in/corner
```

```sh [pnpm]
pnpm add @leafer-in/corner
```

```sh [yarn]
yarn add @leafer-in/corner
```

```sh [bun]
bun add @leafer-in/corner
```

:::

或通过 script 标签引入，使用全局变量 LeaferIN.corner 访问插件内部功能。

::: code-group

```html [corner.min]
<script src="https://unpkg.com/@leafer-in/corner@2.0.9/dist/corner.min.js"></script>
```

```html [corner]
<script src="https://unpkg.com/@leafer-in/corner@2.0.9/dist/corner.js"></script>
```

<!-- https://unpkg.com 无法访问时，可替换为 https://cdn.jsdelivr.net/npm -->

:::
