# scale-fixed

固定元素的全局缩放，不随视图放大。

## 安装插件

需要安装 scale-fixed 插件才能使用，[点此访问 Github 仓库](https://github.com/leaferjs/leafer-in/tree/main/packages/scale-fixed)。

::: code-group

```sh [npm]
npm install @leafer-in/scale-fixed
```

```sh [pnpm]
pnpm add @leafer-in/scale-fixed
```

```sh [yarn]
yarn add @leafer-in/scale-fixed
```

```sh [bun]
bun add @leafer-in/scale-fixed
```

:::

或通过 script 标签引入，使用全局变量 LeaferIN.scaleFixed 访问插件内部功能。

::: code-group

```html [scale-fixed.min]
<script src="https://unpkg.com/@leafer-in/scale-fixed@2.0.9/dist/scale-fixed.min.js"></script>
```

```html [scale-fixed]
<script src="https://unpkg.com/@leafer-in/scale-fixed@2.0.9/dist/scale-fixed.js"></script>
```

<!-- https://unpkg.com 无法访问时，可替换为 https://cdn.jsdelivr.net/npm -->

:::
