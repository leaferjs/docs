# EditorMoveEvent

编辑器的移动事件，通过 `app.editor.on()` 监听。

## 事件属性

### moveX: `number`

X 轴移动距离（世界坐标）

### moveY: `number`

Y 轴移动距离（世界坐标）

## 事件名称

### EditorMoveEvent.MOVE

移动元素事件。

`editor.move`
