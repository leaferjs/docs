# 开发计划

我们正处在一个不断进化完善的过程中，也许现在还缺少一些你需要的功能，请别着急。

以下是我们计划开发的功能列表，同时我们也已经开放了插件机制，大家可以自由发挥、自给自足，从而满足各类应用场景。

::: tip
如果你想更快的拥有这些功能，有两种方式可以加速：

1. 为我们提供赞助，加速我们的发展。

2. 通过我们定义好的接口开发插件（部分功能支持）。

:::
