# Paint

[`SolidPaint`](../../UI/paint/solid.md) | [`LinearGradient`](../../UI/paint/linear.md) | [`RadialGradient`](../../UI/paint/radial.md) | [`AngularGradient`](../../UI/paint/angular.md) | [`ImagePaint`](../../UI/paint/image.md)

填充对象，支持[纯色](../../UI/paint/solid.md)、 [线性渐变](../../UI/paint/linear.md)、[径向渐变](../../UI/paint/radial.md)、[角度渐变](../../UI/paint/angular.md)、[图案填充](../../UI/paint/image.md) 等类型，可用于填充、描边等。

```ts
type Paint =
  | SolidPaint
  | LinearGradient
  | RadialGradient
  | AngularGradient
  | ImagePaint
```
