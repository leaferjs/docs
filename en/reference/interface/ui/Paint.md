# Paint

[`SolidPaint`](../../UI/paint/solid.md) | [`LinearGradient`](../../UI/paint/linear.md) | [`RadialGradient`](../../UI/paint/radial.md) | [`AngularGradient`](../../UI/paint/angular.md) | [`ImagePaint`](../../UI/paint/image.md)

Fill object, supports types such as [solid color](../../UI/paint/solid.md), [linear gradient](../../UI/paint/linear.md), [radial gradient](../../UI/paint/radial.md), [angular gradient](../../UI/paint/angular.md), and [pattern fill](../../UI/paint/image.md). Can be used for fill, stroke, etc.

```ts
type Paint =
  | SolidPaint
  | LinearGradient
  | RadialGradient
  | AngularGradient
  | ImagePaint
```
