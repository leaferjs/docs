# getMotionTotal

The total length of a motion path.

It can be used together with [motionPath](./motionPath.md), [motion](./motion.md), and [getMotionPoint()](./getMotionPoint.md).

:::tip Note
The [Motion Path Plugin](../../plugin/in/motion-path/index.md) must be installed to use this feature.
:::

## Key Method

### getMotionTotal ( ): `number`

Gets the total length of the motion path.

Both elements and motion path elements support this method.
