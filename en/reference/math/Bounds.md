# Bounds

Bounds boundary class

## Key Properties

### x: `number`

x-axis coordinate.

### y: `number`

y-axis coordinate.

### width: `number`

Width, may be negative.

### height: `number`

Height, may be negative.

## Readonly Properties

### minX: `number`

Minimum x-axis coordinate.

### minY: `number`

Minimum y-axis coordinate.

### maxX: `number`

Maximum x-axis coordinate.

### maxY: `number`

Maximum y-axis coordinate.

## Key Methods

When returning [`Bounds`](../../api/classes/Bounds.md), chained operations are supported.

### set ( x: `number` | [`IBoundsData`](../../api/interfaces/IBoundsData.md) = 0, y = 0, width = 0, height = 0 ): [`Bounds`](../../api/classes/Bounds.md)

Set boundary data, same parameters as new Bounds(), default is 0,0,0,0.

### get ( ): [`IBoundsData`](../../api/interfaces/IBoundsData.md)

Get boundary data object {x,y,width,height}.

### clone ( ): [`Bounds`](../../api/classes/Bounds.md)

Clone a Bounds object.

### put ( put: [`IBoundsData`](../../api/interfaces/IBoundsData.md), align: [`IAlign`](../../api/modules.md#ialign) = 'center', putScale?: `IPutScale` )

Place a bounding box into a specified position, and automatically adjust the position and size of put after alignment and scaling.

align represents the alignment position, default is centered placement, putScale represents scaling size, default is 1.

![方向图](/svg/deriction.svg)

```ts
// Direction
type IAlign =
  | 'top-left'
  | 'top'
  | 'top-right'
  | 'right'
  | 'bottom-right'
  | 'bottom'
  | 'bottom-left'
  | 'left'
  | 'center'
```

```ts
type IPutScale =
  | number // specify scale ratio
  | 'cover' // cover (similar to background-size: cover)
  | 'fit' // fit (similar to background-size: contain)
```

### Scaling

### scale ( scaleX: `number`, scaleY = scaleX, onlySize?: `boolean` ): [`Bounds`](../../api/classes/Bounds.md)

Scale the current Bounds around the origin (0, 0).

When onlySize is true, only width and height are scaled.

### scaleOf ( origin: [`IPointData`](../../api/interfaces/IPointData.md), scaleX: `number`, scaleY = scaleX ): [`Bounds`](../../api/classes/Bounds.md)

Scale the current Bounds around the origin point.

### Adjustment

### spread ( fourNumber: [IFourNumber](../interface/math/Math.md#ifournumber) ): [`Bounds`](../../api/classes/Bounds.md)

Expand the boundary outward.

### shrink ( fourNumber: [IFourNumber](../interface/math/Math.md#ifournumber) ): [`Bounds`](../../api/classes/Bounds.md)

Shrink the boundary inward.

### ceil ( ): [`Bounds`](../../api/classes/Bounds.md)

Round up the boundary so that x, y, width, height all become integers, ensuring it still contains the previous Bounds.

### unsign ( ): [`Bounds`](../../api/classes/Bounds.md)

Convert width and height that may be negative into positive values, affecting x and y.

### Add

### add ( bounds:[`IBoundsData`](../../api/interfaces/IBoundsData.md) ): [`Bounds`](../../api/classes/Bounds.md)

Add a bounds.

### addList ( boundsList: [`IBoundsData`](../../api/interfaces/IBoundsData.md)[] ): [`Bounds`](../../api/classes/Bounds.md)

Add a list of bounds.

### setList ( boundsList: [`IBoundsData`](../../api/interfaces/IBoundsData.md)[] ): [`Bounds`](../../api/classes/Bounds.md)

Set boundary data from a bounds list.

### addListWithFn ( list: [`IObject`](../../api/interfaces/IObject.md)[], boundsDataFn: [`IBoundsDataFn`](../../api/interfaces/IBoundsDataFn.md) ): [`Bounds`](../../api/classes/Bounds.md)

Add a bounds list via traversal function. When iterating list, the actual BoundsData is returned by calling boundsDataFn.
:::tip Usage scenario
The list item itself is not Bounds, but one of its properties is Bounds.
:::

### setListWithFn ( list: [`IObject`](../../api/interfaces/IObject.md)[], boundsDataFn: [`IBoundsDataFn`](../../api/interfaces/IBoundsDataFn.md) ): [`Bounds`](../../api/classes/Bounds.md)

Set boundary data from a bounds list generated by a traversal function.

### Coordinates

### setPoint ( point: [`IPointData`](../../api/interfaces/IPointData.md)): [`Bounds`](../../api/classes/Bounds.md)

Set boundary data from a coordinate point, width and height will be 0.

### addPoint ( point: [`IPointData`](../../api/interfaces/IPointData.md)): [`Bounds`](../../api/classes/Bounds.md)

Add a coordinate point.

### setPoints ( points: [`IPointData`](../../api/interfaces/IPointData.md)[] ): [`Bounds`](../../api/classes/Bounds.md)

Set boundary data from a set of coordinate points.

### getPoints ( ): [`IPointData`](../../api/interfaces/IPointData.md)[]

Get the 4 coordinate points [topLeft, topRight, bottomRight, bottomLeft].

### getPoint ( around: [`IAround`](../UI/around.md), onlyBoxSize?: `boolean`, to?: [`IPointData`](../../api/interfaces/IPointData.md) ): [`IPointData`](../../api/interfaces/IPointData.md)

Get the coordinate of the around point in the bounding box.

onlyBoxSize means excluding the starting coordinate of Bounds, default is false.

If to exists, the result will be assigned to to.

### Collision Detection

### hitPoint ( point: [`IPointData`](../../api/interfaces/IPointData.md), pointMatrix?: [`IMatrixData`](../../api/interfaces/IMatrixData.md) ): `boolean`

Check whether it collides with a coordinate point. The passed point supports temporarily applying a matrix pointMatrix.

### hitRadiusPoint ( point: [`IRadiusPointData`](../../api/interfaces/IRadiusPointData.md), pointMatrix?: [`IMatrixWithScaleData`](../../api/interfaces/IMatrixWithLayoutData.md) ):`boolean`

Check whether it collides with a coordinate point with radius. The passed point supports temporarily applying a matrix pointMatrix (requires scaleX and scaleY), commonly used for cursor picking.

### hit ( bounds: [`IBoundsData`](../../api/interfaces/IBoundsData.md), boundsMatrix?: [`IMatrixData`](../../api/interfaces/IMatrixData.md) ): `boolean`

Check whether it collides with another bounds. The passed bounds supports temporarily applying a matrix boundsMatrix.

### includes ( bounds: [`IBoundsData`](../../api/interfaces/IBoundsData.md), boundsMatrix?: [`IMatrixData`](../../api/interfaces/IMatrixData.md) ): `boolean`

Check whether it contains another bounds. The passed bounds supports temporarily applying a matrix boundsMatrix.

### Intersection

### intersect ( bounds: [`IBoundsData`](../../api/interfaces/IBoundsData.md), boundsMatrix?: [`IMatrixData`](../../api/interfaces/IMatrixData.md) ): [`Bounds`](../../api/classes/Bounds.md)

Perform intersection with another bounds, only keeping the overlapping part. The passed bounds supports temporarily applying a matrix boundsMatrix.

### getIntersect ( bounds: [`IBoundsData`](../../api/interfaces/IBoundsData.md), boundsMatrix?: [`IMatrixData`](../../api/interfaces/IMatrixData.md) ): [`Bounds`](../../api/classes/Bounds.md)

Get the intersection with another bounds. The passed bounds supports temporarily applying a matrix boundsMatrix.

### Check

### isSame ( bounds: [`IBoundsData`](../../api/interfaces/IBoundsData.md) ): `boolean`

Check whether it is equal to another bounds.

### isEmpty ( ): `boolean`

Check whether it is an empty bounds, all properties are 0.

### reset ( ): void

Reset the boundary.

## Matrix Methods

### toOuterOf ( matrix: [`IMatrixData`](../../api/interfaces/IMatrixData.md) , to?: [`IBoundsData`](../../api/interfaces/IBoundsData.md) ): [`Bounds`](../../api/classes/Bounds.md)

Convert to the external [AABB bounding box](../UI/bounds.md) of the matrix (assuming current is internal bounds). If to exists, the result will be assigned to to.

### getFitMatrix ( put: [`IBoundsData`](../../api/interfaces/IBoundsData.md) ): [`IMatrix`](../../api/interfaces/IMatrix.md)

Place another bounds into the current bounds and return an optimal fit matrix.

## Examples

### Create via property values

```ts
import { Bounds } from 'leafer-ui'

const bounds = new Bounds(0, 0, 100, 100)
```

### Create via data object

```ts
const boundsData = { x: 0, y: 0, width: 100, height: 100 }

const bounds = new Bounds(boundsData)
```

### Chained operations

```ts
const bounds = new Bounds()

bounds.set(0, 0, 100, 100).hitPoint({ x: 50, y: 50 }) // true
```
