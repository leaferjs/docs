# LeaferJS Showcases at AI Competition, Million-Follower Blogger "Xiaohua" Wins with a Hilarious Project

> 2024-11-06 "Finishing Touch"

On Programmer’s Day 2024, the “Super Code Factory” competition was grandly held, attracting well-known companies such as Tencent Cloud, Baidu Wenxin Kuaima, and Amazon DeepRacer.

![1024](/image/blog/20241106/640.webp)

As an innovative graphics rendering engine, LeaferJS provided strong technical support to participants, helping them realize creative programming and interactive design. Million-follower blogger “Super Xiaohua” won an award with a hilarious and creative game project, adding plenty of highlights to the competition. A video of his creation process is included at the end of the article to take you back through this exciting event.

![1024](/image/blog/20241106/x.png)

## Tencent Cloud AI Code Assistant Special Session

As one of the highlights of the competition, the Tencent Cloud AI Code Assistant special session attracted a large number of programming enthusiasts and innovators. Participants leveraged AI programming tools and APIs to unleash their creativity and develop efficiently, showcasing their technical capabilities. In this context of creativity and technical challenges, “Finishing Touch” became an ideal theme. With LeaferJS, participants were able to easily implement interactive designs and quickly build creative and interactive projects, fully demonstrating LeaferJS’s great potential in creative programming.

## LeaferJS Powers the “Finishing Touch” Competition Theme

![1024](/image/blog/20241106/mz.webp)

Before the competition, Ms. Zhang Jingjing, Head of Tencent Cloud Cloud Studio, introduced the features of the AI Code Assistant and provided a detailed explanation of the “Finishing Touch” theme, encouraging participants to use the LeaferJS library to create impressive game projects.

As the creator of LeaferJS, I guided participants through hands-on demonstrations on how to develop a “Finishing Touch” mini-game using LeaferJS, covering every step from path drawing and character design to interaction implementation, and provided detailed code examples to help participants better understand and use LeaferJS.

Mr. Wang Shengjie demonstrated how to use the AI Code Assistant to quickly explore new ideas. He introduced its capabilities in code generation and intelligent Q&A, and gave a quick demonstration based on the “Finishing Touch” logic.

## AI Empowers Creative Programming, Outstanding Projects Emerge

During the competition, participants combined Tencent Cloud AI Code Assistant with the LeaferJS library to create many impressive works. The convenience and powerful features of LeaferJS enabled participants to quickly draw graphics and add dynamic effects, bringing many highlights to their projects. Ultimately, the first-prize-winning project stood out with its stunning gameplay experience, capturing the audience’s attention and showcasing LeaferJS’s immense potential in creative programming and interactive design. All outstanding projects have been showcased on the Cloud Studio platform, with links provided at the end of the article.
