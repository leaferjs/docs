# Multi Product Forms

## General Use Case

🚀 If you want to start using it in your product immediately, install [leafer-ui](https://leaferjs.com/ui/guide/install/ui/start.html) (70KB min+gzip, zero dependencies) and begin your exploration.

## 高效绘图场景

For high-performance drawing scenarios, it is recommended to install the lighter [leafer-draw](https://leaferjs.com/ui/guide/install/draw/start.html) (58KB min+gzip).

## Game Development Use Case

For game development, it is recommended to install [leafer-game](https://leaferjs.com/ui/guide/install/game/start.html), which already includes game-related plugins.

## 图形编辑场景

For graphic editing, it is recommended to install [leafer-editor](https://leaferjs.com/ui/guide/install/editor/start.html), which already includes editor-related plugins.

## Full Package

You can also install the full [leafer](https://leaferjs.com/ui/guide/install/leafer/start.md) package, which automatically includes `leafer-ui` and all `@leafer-in/*` plugin packages.

You can reduce bundle size by using [import subpackages](./leafer/start.md#import-subpackages). This allows official example code to be used directly, but you must read the [notes](./leafer/start.md#import-subpackages).
