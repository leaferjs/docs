# scale-fixed

Fixes the global scale of elements so they do not scale with the viewport.

## Install Plugin

You need to install the scale-fixed plugin to use it. [Visit Github repository](https://github.com/leaferjs/leafer-in/tree/main/packages/scale-fixed).

::: code-group

```sh [npm]
npm install @leafer-in/scale-fixed
```

```sh [pnpm]
pnpm add @leafer-in/scale-fixed
```

```sh [yarn]
yarn add @leafer-in/scale-fixed
```

```sh [bun]
bun add @leafer-in/scale-fixed
```

:::

Or include via script tag and access plugin features through the global variable `LeaferIN.scaleFixed`.

::: code-group

```html [scale-fixed.min]
<script src="https://unpkg.com/@leafer-in/scale-fixed@2.0.8/dist/scale-fixed.min.js"></script>
```

```html [scale-fixed]
<script src="https://unpkg.com/@leafer-in/scale-fixed@2.0.8/dist/scale-fixed.js"></script>
```

<!-- https://unpkg.com 无法访问时，可替换为 https://cdn.jsdelivr.net/npm -->

:::
