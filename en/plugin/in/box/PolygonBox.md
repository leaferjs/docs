# PolygonBox Element

An element that combines the features of both [Polygon](../../../reference/display/Polygon.md) and [Box](../../../reference/display/Box.md). It also supports adding child elements inside.

<br/>

::: tip Inheritance
PolygonBox  >  [Polygon](../../../reference/display/Polygon.md), [Box](../../../reference/display/Box.md)  >  [Group](../../../reference/display/Group.md), [Rect](../../../reference/display/Rect.md)   >  [UI](../../../reference/display/UI.md)
:::

::: tip Note
You must install the [box plugin](./index.md) to use this feature.
:::
