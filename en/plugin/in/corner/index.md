# Corner

Path corner rounding.

Supports rounded corners between curves and lines, as well as between line segments and curves.

## Install Plugin

You need to install the corner plugin to use it. [Click here to visit the Github repository](https://github.com/leaferjs/leafer-in/tree/main/packages/corner).

::: code-group

```sh [npm]
npm install @leafer-in/corner
```

```sh [pnpm]
pnpm add @leafer-in/corner
```

```sh [yarn]
yarn add @leafer-in/corner
```

```sh [bun]
bun add @leafer-in/corner
```

:::

Or include via script tag and access plugin features using the global variable `LeaferIN.corner`.

::: code-group

```html [corner.min]
<script src="https://unpkg.com/@leafer-in/corner@2.0.8/dist/corner.min.js"></script>
```

```html [corner]
<script src="https://unpkg.com/@leafer-in/corner@2.0.8/dist/corner.js"></script>
```

<!-- https://unpkg.com 无法访问时，可替换为 https://cdn.jsdelivr.net/npm -->

:::
