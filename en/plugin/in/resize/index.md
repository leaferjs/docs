# resize

Resize the bounding box of elements / group elements.

## Install Plugin

You need to install the resize plugin to use it. [Visit Github repository](https://github.com/leaferjs/leafer-in/tree/main/packages/resize).

::: code-group

```sh [npm]
npm install @leafer-in/resize
```

```sh [pnpm]
pnpm add @leafer-in/resize
```

```sh [yarn]
yarn add @leafer-in/resize
```

```sh [bun]
bun add @leafer-in/resize
```

:::

Or include via script tag and access plugin features through the global variable `LeaferIN.resize`.

::: code-group

```html [resize.min]
<script src="https://unpkg.com/@leafer-in/resize@2.0.8/dist/resize.min.js"></script>
```

```html [resize]
<script src="https://unpkg.com/@leafer-in/resize@2.0.8/dist/resize.js"></script>
```

<!-- https://unpkg.com 无法访问时，可替换为 https://cdn.jsdelivr.net/npm -->

:::
