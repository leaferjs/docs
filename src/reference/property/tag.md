# tag

元素标签名（即元素的类名），如 `Rect` 、 `Box`，可通过 [find()](/reference/property/find.md) / [findOne()](/reference/property/findOne.md) 查找。

未来可用于像 HTML 的标签方式创建元素。

## 只读属性

### tag: `string`

元素标签名（即元素的类名）。
