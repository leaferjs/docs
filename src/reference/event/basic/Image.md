# ImageEvent

图片事件。

## 关键属性

### image: [`ILeaferImage`](/api/interfaces/ILeaferImage.md)

原始图片封装对象。

### error: `string` | `object`

图片加载错误信息。

## 辅助属性

用于区分图形内使用多个图案填充的情况。

### attrName: `string`

图案填充的属性名: `fill` | `stroke`

### attrValue: [`ImagePaint`](/reference/property/paint/image.md)

图案填充的属性值。

## 事件名称

### ImageEvent.LOAD

图片开始加载。

`image.load`

### ImageEvent.LOADED

图片加载完成。

`image.loaded`

### ImageEvent.ERROR

图片加载失败。

`image.error`
