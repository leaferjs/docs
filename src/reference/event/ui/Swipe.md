`swipe.left`

### SwipeEvent.RIGHT

向右滑动事件。

`swipe.right`

### SwipeEvent.UP

向上滑动事件。

`swipe.up`

### SwipeEvent.DOWN

向下滑动事件。

`swipe.down`
