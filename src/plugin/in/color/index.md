# color 插件

调整元素/容器的包围盒大小。

## 适用平台

支持所有平台。

## 安装插件

需要安装 resize 插件才能使用，[点此访问 Github 仓库](https://github.com/leaferjs/leafer-in/tree/main/packages/resize)。

::: code-group

```sh[npm]
npm install @leafer-in/resize
```

```sh[pnpm]
pnpm add @leafer-in/resize
```

```sh[yarn]
yarn add @leafer-in/resize
```

```sh[bun]
bun add @leafer-in/resize
```

:::

### 通过 script 标签引入

通过全局变量 LeaferIN.resize 访问插件内部功能。
::: code-group

```html [resize.min]
<script src="https://unpkg.com/@leafer-in/resize@1.0.9/dist/resize.min.js"></script>
```

```html [resize]
<script src="https://unpkg.com/@leafer-in/resize@1.0.9/dist/resize.js"></script>
```

https://unpkg.com 无法访问时，可替换为 https://cdn.jsdelivr.net/npm
:::
